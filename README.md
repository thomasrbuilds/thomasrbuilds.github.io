# Thomas Rule — Finance, Trading Research & Automation

Source for [thomasrbuilds.github.io](https://thomasrbuilds.github.io), my public portfolio of reproducible investment research, portfolio-risk models and financial automation projects.

## Featured work

- [Investment Research Operating System](https://github.com/thomasrbuilds/investment-research-operating-system) — SEC and FRED ingestion, thesis controls, alerts, valuation, decision journaling and weekly research reporting.
- [Portfolio Risk & Stress-Testing Engine](https://github.com/thomasrbuilds/portfolio-risk-stress-testing-engine) — volatility, drawdown, correlation, VaR, Expected Shortfall, risk contribution and scenario analysis.
- [Oracle AI Infrastructure Investment Case](https://github.com/thomasrbuilds/oracle-ai-infrastructure-investment-case) — reproducible analysis of growth, capex, leverage, dilution and execution risk.
- [IBM Investment Thesis](https://github.com/thomasrbuilds/ibm-investment-thesis) — fundamentals, seasonality, valuation and a falsifiable entry framework.

## Site

The portfolio is intentionally lightweight and built with semantic HTML, responsive CSS and a small amount of JavaScript. Project cards link directly to the underlying public evidence and repositories.

To preview locally, open `index.html` in a browser. No build step is required.
